# Git / GitHub 基本メモ

## 今日やったこと
- GitHub用の作業フォルダを作成した
- `git init` でGit管理を開始した
- `README.md` を作成した
- `git add` で変更を登録した
- `git commit` で変更履歴を保存した
- `git remote add origin` でGitHubリポジトリと接続した
- `git push` でGitHubにアップロードした

## Gitとは
ファイルの変更履歴を管理するためのツール。

## GitHubとは
Gitで管理したファイルをオンライン上に保存・共有できるサービス。

## 基本の流れ
1. ファイルを作成・編集する
2. `git status` で状態を確認する
3. `git add .` で変更を登録する
4. `git commit -m "メッセージ"` で変更履歴を保存する
5. `git push` でGitHubに反映する

## よく使うコマンド

### 状態確認

git status
