# Network Security Lab

## 概要
ネットワーク・セキュリティ分野の基礎技術を、実際に手を動かしながら学習した記録です。

## 目的
- Linux環境でネットワークコマンドを扱えるようになる
- TCP/IP、DNS、HTTPなどの通信の流れを理解する
- tcpdumpやWiresharkで通信を観察する
- セキュリティログや攻撃検知の基礎を学ぶ

## 学習予定
- 2026年6月：Git/GitHub整備、Linux/NW基礎の整理
- 2026年7月：tcpdump / Wiresharkによる通信解析
- 2026年8月：nmap、SSH、Firewall、ログ確認
- 2026年9月：仮想ネットワーク構築
- 2026年10月：Snort / Suricataによる簡易IDS構築
